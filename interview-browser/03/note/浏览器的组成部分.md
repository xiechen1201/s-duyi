# 浏览器的组成部分

什么是浏览器？

浏览器主要的组件（不只）：

- 用户界面

- 浏览器引擎

- 渲染引擎

- 网络

- JavaScript 引擎

- 用户界面后端

- 数据存储

## 用户界面

## 浏览器引擎

起到一个桥梁的作用。接收用户界面传递过来的指令，并传递给渲染引擎。

内部工作也是非常复杂的。简单了解浏览器引擎的工作即可。

## 渲染引擎 

负责浏览器请求得到的内容。

## 网络

负责浏览器内各种网络通信。

## JS 解析器

用于解析 JS 脚本的。

谷歌的V8 引擎最出名。

## 用户界面后端

绘制基本窗口小部件。比如输入框、按钮、状态栏等。都是操作系统提供的。

也是一个中间商的作用。

## 数据存储

存储浏览器中的数据。比如 cookie、localstorage、sessionstorage 等。
